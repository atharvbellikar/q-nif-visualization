# Quantum Neural Implicit Field (Q-NIF) - Visualization Pipeline

**Official visualization code for the Springer paper**
*"Quantum-Inspired Neuromorphic FOD Detection: Comprehensive Analysis with Multi-Dataset Evaluation"*

## Features

* One file per Q-NIF module (modular & clean)
* Automatic "Before vs After" detection images
* Exact styling used in the paper examples
* Ready for GitHub README, paper, and supplementary material

## Quick Start

```bash
pip install -r requirements.txt
python run.py
```
